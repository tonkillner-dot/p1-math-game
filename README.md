# คู่มือการใช้งานเกมคณิตศาสตร์ ป.1: ส่วนย่อย-ส่วนรวม (Pixel Math: Part-Whole Adventure)

ยินดีต้อนรับคุณครูเข้าสู่ระบบเกมการศึกษาเรื่อง **ส่วนย่อย ส่วนรวม (Part-Whole Relations)** สำหรับนักเรียนชั้นประถมศึกษาปีที่ 1 เกมนี้ได้รับการออกแบบในสไตล์ย้อนยุค 8-bit (Pixel Punk) ที่สวยงาม ทันสมัย และสามารถควบคุมการเล่นได้ผ่านกล้องเว็บแคมโดยการขยับมือในอากาศ

---

## 🎮 วิธีการเล่นเกม (How to Play)

เกมนี้ประกอบด้วย 2 โหมดหลักในการเรียนรู้คณิตศาสตร์:

1. **โหมดแยกส่วนย่อย (Find the Parts)**
   * โจทย์จะกำหนดจำนวนเต็มมาให้ในกล่อง "ส่วนรวม" (ด้านบน)
   * ให้นักเรียนแบ่งจำนวนนั้นออกเป็นสองกลุ่มย่อยลงในกล่อง "ส่วนย่อย A" และ "ส่วนย่อย B" ด้านล่างให้ถูกต้อง
   * *ตัวอย่าง*: ส่วนรวมคือ 5 นักเรียนสามารถแบ่งเป็น ส่วนย่อย A = 2 และ ส่วนย่อย B = 3

2. **โหมดรวมเป็นส่วนรวม (Find the Whole)**
   * โจทย์จะระบุจำนวนสิ่งของในกล่อง "ส่วนย่อย A" และ "ส่วนย่อย B" ด้านล่าง
   * ให้นักเรียนรวมจำนวนทั้งสองกลุ่ม แล้วกรอกคำตอบที่ถูกต้องลงในกล่อง "ส่วนรวม" ด้านบน
   * *ตัวอย่าง*: ส่วนย่อย A = 2, ส่วนย่อย B = 3 ดังนั้นส่วนรวมต้องปรับค่าเป็น 5

## 📷 วิธีการควบคุมด้วยเว็บแคม (Webcam Controls)

ในเมนูหน้าแรก คุณครูสามารถเปิดสวิตช์ **"ใช้การกวักมือผ่านกล้อง (Webcam Mode)"** เพื่อเข้าสู่โหมดควบคุมไร้การสัมผัส

1. เมื่อเริ่มเล่นเกม หน้าจอแสดงวิดีโอจากกล้องจะปรากฏขึ้นพร้อมกล่องสี่เหลี่ยมตรวจจับ 2 ช่อง:
   * **ช่องซ้าย (PART A)**
   * **ช่องขวา (PART B)**
2. **การปรับเปลี่ยนจำนวน**:
   * หากต้องการเพิ่มจำนวนของกลุ่มย่อย A ให้นักเรียนยกมือโบกพัดเบาๆ ในโซนกล่องฝั่งซ้ายของจอภาพ
   * หากต้องการเพิ่มจำนวนของกลุ่มย่อย B ให้นักเรียนยกมือโบกพัดเบาๆ ในโซนกล่องฝั่งขวาของจอภาพ
   * ค่าตัวเลขจะเพิ่มขึ้นทีละ 1 เมื่อตรวจจับความเคลื่อนไหวได้ และหากค่าเลยจำนวนสูงสุด ระบบจะวนกลับไปที่ 0 อัตโนมัติ เพื่อให้เด็กๆ เล่นง่ายและไม่สับสน
3. **ระบบปุ่มสำรอง (Fallback)**:
   * นักเรียนสามารถกดปุ่มเครื่องหมาย `+` หรือ `-` ด้านล่างของกล่องคะแนนได้ด้วยการคลิกเมาส์หรือแตะที่หน้าจอสัมผัสได้ทุกเวลา หากไม่สะดวกใช้งานกล้อง

## 📁 โครงสร้างระบบไฟล์ในโปรเจกต์ (Project Structure)

โปรเจกต์นี้ได้รับการพัฒนาขึ้นโดยใช้หน้าเว็บแบบ Static Web App ปราศจากความจำเป็นในการใช้เฟรมเวิร์กขนาดใหญ่ เพื่อให้ง่ายที่สุดต่อการเผยแพร่และไม่เกิดข้อผิดพลาดในการโหลดไฟล์เสียง

* `index.html` — โครงสร้างเมนู หน้าเล่นเกม และส่วนประกอบหลัก
* `style.css` — การแต่งสไตล์เรโทร 8-bit Pixel Art, Scanlines, และ CRT TV Effect
* `game.js` — ตรรกะคณิตศาสตร์ ระบบสุ่มโจทย์ และตรวจสอบผลคะแนน
* `motion.js` — โปรแกรมการประมวลผลพิกเซลจากวิดีโอกล้องเว็บแคมเพื่อจับความเคลื่อนไหว
* `audio.js` — ระบบสังเคราะห์เสียงบี๊บ 8-bit ด้วยเทคโนโลยี Web Audio API
* `vercel.json` — ไฟล์การตั้งค่าประสิทธิภาพและการรักษาความปลอดภัยสำหรับ Vercel
* `README.md` — เอกสารแนะนำเล่มนี้

## 🚀 วิธีการอัปโหลดขึ้น GitHub และ Deploy บน Vercel

คุณครูสามารถนำไฟล์ทั้งหมดขึ้นออนไลน์เพื่อให้เด็กๆ เล่นผ่านอินเทอร์เน็ตได้ฟรีตามขั้นตอนต่อไปนี้ครับ:

### ขั้นตอนที่ 1: การนำรหัสโปรเจกต์ขึ้น GitHub
1. สมัครใช้งานหรือลงชื่อเข้าใช้ที่เว็บ [GitHub.com](https://github.com/)
2. กดสร้าง Repository ใหม่ (New Repository) ตั้งชื่อ เช่น `p1-math-game`
3. ทำการอัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้นไปบนคลังโปรเจกต์ (หรือใช้คำสั่ง Git Push หากคุณครูชำนาญการเขียนโค้ด)

### ขั้นตอนที่ 2: การเปิดใช้งาน (Deploy) บน Vercel
1. สมัครใช้งานเว็บ [Vercel.com](https://vercel.com/) (แนะนำให้ล็อกอินด้วยบัญชี GitHub เพื่อความสะดวก)
2. เลือกกดปุ่ม **"Add New Project"** จากหน้า Dashboard
3. ค้นหาคลังเก็บของ `p1-math-game` จาก GitHub ที่คุณครูอัปโหลดไว้แล้วกดปุ่ม **"Import"**
4. Vercel จะตั้งค่าระบบ Static Site ให้อัตโนมัติ ให้คุณครูกดปุ่ม **"Deploy"** ได้ทันที
5. รอไม่เกิน 1 นาที ระบบจะให้ที่อยู่ลิงก์เว็บสำเร็จรูป (เช่น `p1-math-game.vercel.app`) เพื่อนำไปให้นักเรียนเปิดเล่นบนคอมพิวเตอร์ แท็บเล็ต หรือสมาร์ทโฟนที่มีกล้องหน้าได้ทันทีครับ!
